# 📚 StudyFlow

## 📌 Descrição
O StudyFlow é uma aplicação web full stack de gerenciamento de estudos e produtividade acadêmica. Permite organizar sessões de estudo, acompanhar progresso e validar a qualidade do sistema por meio de testes unitários automatizados.

## 🚀 Tecnologias

### Backend
- Node.js
- Express
- JavaScript
- MySQL

### Frontend
- React
- Vite

### Testes
- Jest (backend)
- Playwright (frontend)

## ⚙️ Funcionalidades
- Cadastro, listagem, atualização e remoção de usuários
- Criação e gerenciamento de sessões de estudo
- Controle de progresso acadêmico
- Testes unitários no backend e frontend

## 🧪 Testes
O projeto conta com testes automatizados cobrindo frontend e backend:

- **Jest** — testes unitários no backend (repository, service, controller)
- **Playwright** — testes end-to-end no frontend (simulação de interações do usuário)

## 🧠 Objetivo
Projeto desenvolvido para praticar desenvolvimento full stack, integração com banco de dados relacional e aplicação de testes de software conforme práticas da indústria.

## ▶️ Como rodar o projeto

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
