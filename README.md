# 📚 StudyFlow

## 📌 Descrição
O StudyFlow é uma aplicação para gerenciamento de estudos, permitindo organizar sessões de estudo, acompanhar o progresso e melhorar a produtividade.

## 🚀 Tecnologias
- TypeScript
- Node.js
- Prisma
- Express

## ⚙️ Funcionalidades
- Cadastro de usuários
- Criação de sessões de estudo
- Listagem de sessões
- Observar desempenho entre sessões
- Atualização e remoção de dados

## 🧠 Objetivo
Este projeto foi desenvolvido para praticar desenvolvimento full stack e capacidade de aplicar testes de funcionalidades em sistemas.
